# Walkthrough - Premium Developer Portfolio Website

We have successfully designed and built a premium, award-winning personal portfolio website tailored for **Vansh** (Full Stack MERN Developer). The project utilizes a modern stack featuring **React, Vite, Tailwind CSS v4, GSAP, Framer Motion, Lenis Smooth Scroll, and Swiper.js**.

---

## 🎨 Design System & Aesthetics
- **Theme**: Apple-grade minimalism combined with modern glassmorphic overlays.
- **Color Palette**: Light Background (`#F8FAFC`), Dark Background (`#0F172A`), Primary Sky Blue (`#2D8CFF`), Secondary Light Blue (`#60A5FA`), Accent Deep Blue (`#2563EB`).
- **Typography**: Display Headers set to **Clash Display** & **Satoshi** via Fontshare. Body text set to **Inter** via Google Fonts. Custom handwritten signature uses **Alex Brush**.
